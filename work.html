I have done several projects that reflect my technical skills and creativity. Here is a selection of my most significant works.

CinemaApp

CinemaApp is a Windows Forms application designed to manage a list of movie movies. It uses a FilmController class to coordinate movie management operations, allowing users to add new movies, edit existing details, delete movies, and view selected movie details. The application also provides functionality to load and save the list of movies in JSON files to maintain data between sessions.

ListGraphApp

ListGraphApp is a Windows Forms application designed to demonstrate the implementation of generic list and graph data structures. It provides CRUD (Create, Read, Update, Delete) operations for managing lists of integers or strings and graphs with nodes containing integer or string values.
